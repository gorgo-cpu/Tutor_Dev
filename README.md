# TutoringDE
Die website um Lukas's Business Idee aufzubauen 

HauptIdee

  Eine Website die nur für's Deutsch Tutoring gedacht ist.
  3 Haupt-Landingpages und zwar eine für Eltern eine für Schülern und eine für Lehrer.
  MVP Page mit Beschreibung des Projekts und eine möglichkeit sich einzuloggen 
  3D low Poly Elemente mit denen man interagieren kann
  selber hosten auf einen lokalen Server 
  

Was wir benutzen um dies zu verwirklichen 

  R3F to implement and animate 3D objects on the Frontend 
  some otel collector 
  Postgresql or some variation of SQL for the database engine
  keycloak for security
  rabbit mq to orchestrate workflows and make sure everything is communicating correctly
  and docker backend, Kotlin/Springboot application to run the locall server 

(INFO alle oben genannten resourcen um dieses Projekt aufzubauen sind open-source und 
kostenlos nutzbar, ob es alternativen gibt die besser sind und ein Abo benötigen wird 
nach dem ersten funktionellen Prototyp des Projekts besprochen werden. )
